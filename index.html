Jindori 챗봇 사용법

우리 팀 전용 디스코드 봇의 주요 기능과 명령어를 확인해보세요!

💬 번역 명령어

  • /activate-translation - 채널에 자동 번역 기능을 활성화합니다. 번역 기능 활성화 시, 모든 메시지에 대해 자동으로 번역이 제공됩니다.
  • /deactivate-translation - 채널의 자동 번역 기능을 비활성화합니다.

⏰ 알람 명령어

  • /list-alarm - 채널에 등록된 알람 목록을 확인합니다. 각 알람에는 고유 ID가 부여되며, 해당 ID를 통해 수정 또는 삭제할 수 있습니다.
  • /alarm - 특정 시간에 채널에 메시지를 전송하도록 알람을 설정합니다.
    명령어 사용법 및 옵션
    • date: yyyy-mm-dd 형식의 날짜 (필수)
    • time: hh:mm 형식의 시간 (필수)
    • target_user: 알람 대상 유저 (필수)
    • message: 전송할 메시지 (필수)
    • recurring_option: 반복 주기 선택 (선택)
      • once - 한 번만
      • hourly - 매시간
      • daily - 매일
      • weekly - 매주
      • monthly - 매월
      • yearly - 매년
  • /delete-alarm - 알람 ID를 기반으로 등록된 알람을 삭제합니다. `/list-alarm` 명령으로 확인한 `alarm_id`를 입력하여 해당 알람을 제거할 수 있습니다.
